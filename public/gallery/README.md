# Gallery Image Names

Add gallery photos to this folder using these exact names:

1. `hana-gallery-01.jpg` - Hana Portrait
2. `hana-gallery-02.jpg` - Debut Details
3. `IMG_3290.jpeg` - Family Moment
4. `IMG_4700.jpg` - Golden Glow

Use the exact capitalization shown here. If you want to use different file names, update the matching file paths in `components/GallerySection.js`.
